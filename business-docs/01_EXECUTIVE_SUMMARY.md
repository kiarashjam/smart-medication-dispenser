### How It Works

```
┌─────────────────────────────────────────────────────────────────────┐
│                         PATIENT JOURNEY                              │
├──────────┬──────────┬──────────┬──────────┬──────────┬──────────────┤
│ Schedule │ Dispense │  Alert   │ Confirm  │  Track   │   Report     │
│ set by   │ at right │ via app  │ patient  │ history  │ to caregiver │
│ caregiver│ time     │ & device │ took meds│ & trends │ in real-time │
└──────────┴──────────┴──────────┴──────────┴──────────┴──────────────┘
```

1. **Caregiver sets schedule** via web portal or app
2. **Device dispenses** at scheduled time with audio/visual alert
3. **Patient notified** via app push notification and device alert
4. **Patient takes medication** — sensor confirms removal
5. **Caregiver sees confirmation** in real-time
6. **If missed** — escalating alerts to patient, then caregiver
